Examples

  1. Reference jQuery <script src="jquery-1.8.3.min.js"></script>
  2. Reference inlineIpsum <script src="jquery.inlineIpsum.min.js"></script>
  3. Initialize the plugin <script>$('body *').inlineIpsum();></script>
  4. Use Razor like syntax to trigger inlineIpsum, any @Html.Ipsum code found within the body of a document will be replaced with Lorem Ipsum text.
  5. Basic HTML elements, filled with Lorem Ipsum are created by using the simple fluent api @Html.Ipsum.h1().p().h2().p()
  6. Take full control over the output by passing arguements @Html.Ipsum.p(paragraphs, sentences, htmlAttributes) See the examples below for more details.

Paragraph

This is using seedrandom to always stay the same.

Code: @Html.Ipsum.seedRandom("seed").p()
Result: @Html.Ipsum.seedRandom("seed").p()

Four paragraphs, ten sentences

This one will use seedrandom without a seed to set itself back.

Code: @Html.seedRandom().Ipsum.p(4, 10)
Result: @Html.Ipsum.seedRandom().p(4, 10)

Paragraph with the attribute class="fancy"

Code: @Html.Ipsum.p(1,10,{ "class":"fancy" })
Result: @Html.Ipsum.p(1,10,{ "class":"fancy" })

h1 tag

Code: @Html.Ipsum.h1()
Result: @Html.Ipsum.h1()

h1 tag, just one word long

Code: @Html.Ipsum.h1(1)
Result: @Html.Ipsum.h1(1)

h2 tag, with the attribute data-special="true"

Code: @Html.Ipsum.h2(5, { "data-special": "true" })
Result: @Html.Ipsum.h2(5, { "data-special": "true" })

unordered list

Code: @Html.Ipsum.ul()
Result: @Html.Ipsum.ul()

unordered list of links

Code: @Html.Ipsum.ul(4,2,true)
Result: @Html.Ipsum.ul(4,2,true)

a mock blog post

Code: @Html.Ipsum.blogPost()
Result: @Html.Ipsum.blogPost()

Fluent api

@Html.Ipsum.h1().p().h2().p().h3().ol(10,3, true)
Result: @Html.Ipsum.h1().p().h2().p().h3().ol(10,3, true)
